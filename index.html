Html常用标签

HTML 常用标签 一.a 标签的用法 属性： href:添加链接 target：指定在哪个页面打开超链接 download：下载页面 rel=noopener 作用：1、跳转到外部页面 2、跳转内部锚点 3、跳转到邮箱或者电话 a 标签 href 的取值： 1、网址：例如http://google.com 2、路径 3、伪协议：例如 JavaScript、mailto、tel 二、table 标签 table标签用来定义 HTML 表格。 简单的 HTML 表格由 table 元素以及一个或多个 tr、th 或 td 元素组成。 tr 元素定义表格行，th 元素定义表头，td 元素定义表格单元。 table标签里面通常只有三个标签：thead、tbody、tfoot 三、image 标签 1、作用 发出get请求，展示一张图片 2、属性 alt：加载失败时，显示提示 heigh：高度 width：宽度 src：图片来源 3、事件 onload：加载成功... onerror：加载失败，提示... 四、感悟 这一节学习了 html 几种常用标签的用法，可以在 html 文件 里面插入网址、表格、图片、电子邮箱等等，使得网页的实 用性进一步加强，感觉收获满满，让我对接下面的学习充满期待，希望能早日写出漂亮实用的网页，Fighting！

Html入门笔记

HTML基本标签 一、html的发明 html标准制定者是W3C，由万维网发明者Tim Berners-Lee创立 二、html起手式 三、常用的章节标签 标题 h1~h6 章节 section 文章 article 段落 p 头部 header 脚部 footer 主要内容 main 旁支内容 aside 划分 div 四、全局属性 class 用来对网页元素进行分类，如果不同元素的chass属性值相同，就表示他们是同一类的。 contenteditable 使元素变得可编辑 hidden 使元素隐藏，不可见 id 此属性是元素在网页内的唯一标识符，网页可能包含多个标签，id属性可以指定每个标签的唯一标识符。 style 此属性用于指定当前元素的css样式。 tabindex 使浏览器允许使用tab键遍历网页元素。 title 用来为元素添加附加说明 五、内容标签 ol+li ol指的是有顺序的列表，li指的是列表中的一项 ul+li 和上面的相反，无顺序的列表 dl+dt+dd dl：描述列表，dt：被描述的项目，dd：描述项目的内容。 pre 对选中部分保留换行和空格 code：将其中的字体设置成代码样式（等宽） hr：加分割线 br：换行 a：插入链接，一般格式为：<a href="网址" target="_blank">名称</a> en 强调某一段或某一句，变为特殊字体 strong 使其中的内容加粗 quote 引用 blockquote 换行引用
